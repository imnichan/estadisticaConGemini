# IA-Stats Studio 📊
Aplicación de Análisis Inferencial asistida por Gemini 2.0 Flash.

## Funcionalidades
- Generación de datos sintéticos y carga de CSV.
- Visualización de distribución (KDE & Boxplots).
- Prueba de Hipótesis Z automatizada.
- Interpretación de resultados mediante IA.

## Instalación
1. Clonar el repo.
2. `pip install -r requirements.txt`
3. Configurar `.env` con tu GEMINI_API_KEY.
4. `streamlit run app.py`
